# Legal Notice

## About This Project

This project provides Docker deployment configurations for [Stirling PDF](https://github.com/Stirling-Tools/Stirling-PDF), a powerful PDF manipulation tool.

This repository contains **deployment configurations only** - not a fork or modification of Stirling PDF itself:

- Docker Compose configurations for multiple deployment scenarios
- Environment variable templates and documentation
- Custom branding scripts for white-label deployments
- Certificate management scripts for PDF signing
- German and English documentation
